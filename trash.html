Understanding Vibe Coding

Vibe coding means combining AI-generated code with human understanding: you use AI for suggestions but must test, understand, and adapt the results, not just copy-paste them.

For example, official guidelines warn that "copying AI-generated code without understanding" is unethical.

Understand Vibe Coding

  • Treat AI as an assistant
  • Read and review suggested code
  • Test thoroughly before implementation

Use AI Thoughtfully

  • Verify and review AI-generated code
  • Cross-check logic and performance
  • Compare against official documentation

Learn New Tech Quickly

  • Rely on official documentation and tutorials
  • Use trusted resources:
    • Mozilla MDN for web tech
    • Flask/Django docs
    • Kivy documentation
  • Experiment with small code snippets

Rules (Dos & Don'ts)

Do's

  • Embrace AI Tools and Online Resources: Feel free to use AI tools like ChatGPT or GitHub Copilot, and leverage online learning resources for help and ideas. Just remember to refine and test any code they provide.
  • Collaborate on Team Projects: For your team project, actively work with your teammates. This means dividing tasks, integrating your code, and helping each other out to achieve your shared goal.
  • Work Honestly and Ask Questions: Always complete your assignments with integrity. If you're unsure about any rule, don't hesitate to ask your mentors or instructors for clarification.

Don'ts

  • Avoid Plagiarism: Do not copy entire projects or large chunks of code from others. This includes submitting AI-generated code without understanding it or making modifications.
  • Don't Seek Unauthorized Help: Avoid getting external, unauthorized assistance, such as paid help or direct answers to your project challenges. Cheating will lead to disqualification.
  • Respect Deadlines: Make sure to submit your projects on time. Late submissions will not be accepted unless you've been granted an extension in advance.
  • Maintain Professional Conduct: Throughout the event, behave professionally and show respect to your peers and the judges. Any harassment or disruptive behavior is unacceptable.
